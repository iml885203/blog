常用的電腦硬體測試軟體整理

名子 硬體 目的 備註
3DMARK CPU & GPU & 其他綜合效能 測試、跑分 付費但Steam上常特價
CrystalDiskMark 硬碟 跑分 有排名比較
CrystalDiskInfo 硬碟 檢測 支援SSD
Cinebench R20 CPU 跑分 在微軟商店
CPU-Z CPU 跑分、硬體查看
FurMark GPU 燒機 可測穩定性、散熱
OCCT Power、CPU、GPU、RAM 燒機、溫度查看 測穩定性、散熱，換顯卡 or Power 可測
HWMonitor ALL 溫度查看
CoreTemp CPU 溫度查看
