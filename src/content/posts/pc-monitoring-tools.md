常用的電腦硬體測試軟體整理
| 名子 | 硬體 | 目的 | 備註 |
| --- | --- | --- | --- |
| [3DMARK](https://store.steampowered.com/app/223850/3DMark/) | CPU & GPU & 其他綜合效能 | 測試、跑分 | 付費但Steam上常特價 |  |
| [CrystalDiskMark](https://www.azofreeware.com/2010/01/crystaldiskmark-220.html) | 硬碟 | 跑分 | 有排名比較 |
| [CrystalDiskInfo](https://www.azofreeware.com/2010/01/crystaldiskinfo-330.html) | 硬碟 | 檢測 | 支援SSD |
| [Cinebench](https://www.microsoft.com/zh-tw/p/cinebench/9pgzkjc81q7j) R20 | CPU | 跑分 | 在微軟商店 |
| [CPU-Z](https://www.azofreeware.com/2006/10/cpu-z-137.html) | CPU | 跑分、硬體查看 |  |
| [FurMark](https://www.azofreeware.com/2021/02/furmark.html) | GPU | 燒機 | 可測穩定性、散熱 |
| [OCCT](https://www.azofreeware.com/2016/05/occt.html) | Power、CPU、GPU、RAM | 燒機、溫度查看 | 測穩定性、散熱，換顯卡 or Power 可測 |
| [HWMonitor](https://www.azofreeware.com/2008/03/hwmonitor-1080.html) | ALL | 溫度查看 |  |
| [CoreTemp](https://www.alcpu.com/CoreTemp/) | CPU | 溫度查看 |  |
